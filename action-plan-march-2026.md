# HumanPages — Action Plan
**March 22, 2026**

Core insight: "Try it for free" failed. Devs don't need to hire someone right now. The strategy must shift to **"show the paradigm shift"** — let devs SEE what happens when an AI agent can transact with humans, so they want the MCP when their moment comes.

---

## PHASE 1: Next 24 Hours (before Starkware)

**Goal:** Arm yourself with demo assets + nail the Starkware pitch.

### 1. Record 2-3 killer demo videos (3-4 hours)
- **Demo A** — "Cursor agent hires a designer." Screen-record an AI agent in Cursor browsing HumanPages providers, selecting one, sending a brief, receiving deliverables. Real task, real provider, real result. 60 seconds.
- **Demo B** — "Agent does what agents can't." Show a physical-world task: agent hires someone to verify something on-site, take photos, do a local errand. This is the "whoa" moment — no other MCP can do this.
- **Demo C** — "From prompt to completion." Full loop: dev types a prompt → agent discovers it needs a human → searches providers → hires → task delivered. 90 seconds.
- **Cost:** $15-40 (pay providers for the demo tasks)
- **Output:** 3 short videos ready for Twitter, hackathon, Starkware, all channels

### 2. Prep Starkware pitch (2 hours)
- Lead with Demo A or B — don't open with slides, open with "watch this"
- Frame: "1,500 technical freelancers already onboarded. MCP is live. Here's what it looks like in action."
- Africa angle: "We're enabling African freelancers to be discovered and hired by AI agents globally. Here's the $5-20 task economy we're building."
- Ask: Confirm which grant programs fit. Africa fund is one option — be open to others.
- Prepare a one-pager with: live MCP, 1,500 providers, demo links, 24h/2-week/1-month roadmap

### 3. Post Demo A on Twitter from official account (30 min)
- Caption: "Has your AI agent ever executed a transaction on your behalf?" (your idea #14 — now with video proof)
- This is not a CTA to sign up. It's a "look what's possible" moment.
- Pin the tweet.

### 4. Send demos (not promo links) to engineer friends (1 hour)
- Instead of "Itai-loves-you" link, send the 60-second video with: "Built this. Your Cursor agent can now hire humans. Thought you'd want to see it."
- No ask. No free credits. Just show.
- The ones who reply "how do I try this?" are your real leads.

**Phase 1 budget: ~$50**

## PHASE 2: Next 2 Weeks

**Goal:** Get 30-50 devs to install the MCP by showing use cases in the right contexts.

### 5. Hackathon — synthesis.md (highest priority)
- Don't pitch "use our MCP." Instead: "Your hackathon project can do something nobody else's can — your agent can hire a human mid-flow."
- Play Demo C at your booth/talk. Show the paradigm shift.
- Offer to help 3-5 teams integrate the MCP into their hackathon project. Be hands-on.
- **Budget:** $100 (booth materials + subsidize tasks for teams that integrate)
- **Metric:** 3-5 teams integrate MCP into their project

### 6. Da Nang AI community + NomadFest — live demo (high priority)
- Organize or join a meetup. Run the agent live — hire someone in real-time, show the result.
- 5-minute demo is worth more than any landing page.
- Ask community organizers if you can do a lightning talk.
- **Budget:** $50 (meetup costs + demo task)
- **Metric:** 10-15 devs see it live, 3-5 install MCP

### 7. Twitter content series (ongoing, free)
- Post demo clips 2-3x/week from official account
- Each clip shows a different use case: design task, code review, physical verification, SEO content
- Engage on AI dev posts with the clips, not with promo links
- Comment on Cursor/Claude/AI agent threads: "Here's what this looks like with a human in the loop" + video
- **Budget:** $0
- **Metric:** Track impressions + link clicks via UTM

### 8. SEO subpages — let them work in background (free)
- The /dev/connect/ pages targeting "openclaw" etc. are already built
- Don't spend more time on these now — SEO takes weeks. Let them compound.
- **Budget:** $0

### 9. Hire from own platform for content creation (smart spend)
- Use HumanPages to hire 3-5 freelancers to create: short-form video edits of your demos, social media posts, SEO blog content
- This serves double duty: creates marketing content AND is itself a case study ("we used our own platform to create this")
- **Budget:** $100-150
- **Metric:** 3-5 content pieces produced via MCP

### 10. Reddit — show, don't shill (careful)
- Post demo videos in r/cursor, r/ClaudeAI, r/LocalLLaMA, r/SideProject
- Frame: "I built an MCP that lets AI agents hire humans. Here's a demo." — genuine show-and-tell
- Do NOT astroturf. One authentic post with video > 10 fake upvotes.
- Your people can upvote and comment naturally but don't make it obvious.
- **Budget:** $0
- **Metric:** Post engagement + UTM clicks

### 11. Set up conversion tracking (one-time, day 1 of phase 2)
- UTM parameters on every link: utm_source=twitter/hackathon/reddit/dnang etc.
- Track: page views → MCP installs → first task created → task completed
- Without this you're flying blind. Wire it before anything else in phase 2.
- **Budget:** $0 (your time)

**Phase 2 budget: ~$300-400**

## KEY LESSON FROM "ITAI-LOVES-YOU" FAILURE

Devs don't adopt tools because they're free. They adopt tools when they **see something that changes how they think about what's possible.** Every action in this plan starts with showing, not offering. The promo links and free credits become relevant AFTER someone has had the "whoa" moment — not before.
