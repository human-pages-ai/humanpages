# Human Pages MCP Bundle

This is the `.mcpb` bundle for Claude Desktop.

## Installation

1. Download `humans.mcpb`
2. Double-click to install
3. Restart Claude Desktop

## Manual Installation

If auto-install doesn't work, copy the contents of this bundle to your Claude Desktop MCP servers directory and update your configuration.

## Tools Included

- `search_humans` - Search for humans by skill/location
- `get_human` - Get a human's full profile
- `record_job` - Record job assignments

## Configuration

Set `HUMANS_API_URL` environment variable to point to your API server.
